Buffalo Tech just as bad as Seagate ? HD-LX2.0TU-EU

11 Aug 2012

I just received my 2 TB Buffalo Tech external USB drive (HD-LX2.0TU-EU). I had to buy a new drive because my external 1 TB Seagate drive started to fail. This makes my 3rd (THIRD) Seagate drive which failed and one of those failures costed me $3000 data rescue bill. It was a good lesson to set up an automated incremental backup system thereafter though. That's it, I will NEVER buy a Seagate product ever again. But, wait, Buffalo turns out to be worse! It didn't work from the beginning :)

How do you tell if a drive is failing ?

Simple, by checking S.M.A.R.T. diagnostic values.

If you are lucky enough, you'd have a tool that periodically checks your drives S.M.A.R.T. diagnostics and report you in the case of a possible problem. The most typical S.M.A.R.T. attribute I saw on failing hard drives was Reallocation Count, it means that some sectors on the drive starts malfunctioning hence those sectors are mapped to some spare sectors. This simply means that your hard drive is starting to fail and it will soon collapse. Sorry.

If you were not lucky enough to get these signs before it's too late, you'll start to see ugly face of disk failures.

Apparently (or naturally) most software developers does take it for granted that a disk access will take quick enough time that, it does not need to be done in another "thread". These software developers include the giants such as Microsoft and Apple. Imagine you're trying to open a folder to list its contents in Windows Explorer and the drive is failing, soon enough, whole your Windows Explorer window "freeze", stopping processing UI events, such as mouse or keyboard events. In fact, the software is not frozen, it simply is waiting for the disk read operation to complete, which virtually renders it frozen :)

I do not understand why Microsoft engineers decided to design Windows Explorer in such a manner, though maybe, it's simply me misinterpreting the problem above, maybe the problem lays somewhere else.

Microsoft is not alone, I just witnessed how Apple's Disk Utulity and whole OS UI started to fall apart when a failing drive was trying to be read.

And not just that, our beloved VLC, XBMC they all suffer when the local disk file read attempt is halted. XBMC simply looks like frozen, not processing any UI events.

So what happened with Buffalo Tech External Hard Drive ?

Well, I just received my new hard drive and unboxed it, as we usually do. I connected it to my Mac Mini (latest series) which runs Lion (10.7). Nothing happened, I launched Disk Utulity and everything just got unresponsive, after a while I saw the drive in Disk Utulity, but everything was so unresponsive, it was practically impossible to work with.

Then, I though I should give Windows a try and connected this drive to my Sony VAIO Laptop. Windows popped up the System Tray balloon as it usually does, "Installing driver for new device". It took quite some time, I tried re-plugging the drive (probably a bad idea), then balloon told me that new driver installation failed.

Then I checked Window Device Manager, there it was, USB Mass Storage Device, though after a while, a yellow exclamation mark appears next to it. When you double click the device, Device Status says This device cannot start. (Code 10).

Though the documentation provided by Buffalo says, there should be three devices listed.

Device                                  Device Name in Device Manager
Universal Serial Bus controllers        USB Mass Storage Device
Disk drives                             BUFFALO HD-LXU3 USB Device
DVD/CD-ROM Drive                        BUFFALO Virtual Cdrom USB Device

It turns out that in the first try, it could read some information such as the partition table from the disk, but as you can see the I/O errors (which caused the extreme unresponsiveness in the Mac UI) the drive started failing immediately.

Finally, I thought there's a problem with the USB 2.0 compatibility. This drive is marketed as a super fast USB 3.0 drive with claimed USB 2.0 compatibility. Neither my Mac Mini nor my Sony VAIO have USB 3.0. So, I got my brother new VAIO and tried with its USB 3.0 interface... Again with no success.

In the end this drive is tested with Mac Mini, Macbook and two Sony VAIO laptops with no success.

I've sent an email to Buffalo Tech support, we'll see how they handle this situation.
